# BillPay Secure Mobile

Simple iPhone (and Android) app for BillPay Secure.

## Setup

1. Create `.env` from parent config:

```bash
npm run setup
```

Or manually create `.env` with `EXPO_PUBLIC_SUPABASE_URL`, `EXPO_PUBLIC_SUPABASE_ANON_KEY`, `EXPO_PUBLIC_API_URL`.

2. Install deps and run:

```bash
npm install
npm start
```

3. Scan the QR code with your iPhone camera to open in Expo Go, or run `npm run ios` for simulator.
